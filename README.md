MagikDamage is a packet-based damage indicators plugin for [Paper](https://papermc.io/).

## Features
**Packet-based and asynchronous** MagikDamage is packed-based and asynchronous so it doesn't block the main server thread.<br>
**Per damage type indicatorSettings** You can set a custom indicatorSettings style for each damage type or health regain reason.<br>
**Fully-customizable indicatorSettings** You can customize the text display to fit in your needs.

## Compatibility
MagikDamage is compatible and has been tested with Paper 1.21.4+ and requires Java 21 (or higher). 
May work with older Paper versions.
